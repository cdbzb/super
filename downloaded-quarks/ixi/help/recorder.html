ixiQuarks : Recorder

 

 

Sound Recoder
 

The Sound Recorder records sound onto the hard disk. The file can be found in the ixiquarks folder inside the sounds folder of the application. The popup menu indicates which audio channel is recorded.


The stereo and mono radiobuttons define if the file is stereo or mono. The recorded file is a 44.1 KHz, 16 bit, AIF file.
